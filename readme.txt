Проект для замены китайскго специализированного микропроцессора HC8T1212 использующегося в настольной лампе ЭРА NLED-473 (сенсорная регулировка цвета: теплый(Y)-холодный(W)-белый(YW), регулировка яркости через инкрементный повортный энкодер, включение-выключение).
В проекте использован микропроцессор Attiny24A (может быть заменена на Attiny44A, Attiny84A) и одноканальный драйвер сенсорной клавиатуры AT42QT1010 (может быть заменена на AT42QT1011)
